About War1gus

War1gus is a Warcraft Mod that allows you to play Warcraft with the Stratagus engine. The game looks and sounds exactly like Warcraft, see the screenshot.

Installation

During the installation of War1gus you will be asked for a copy of the original Warcraft, so that the installer can extract the data.

Disclaimer

War1gus is not an official Blizzard product, its a Warcraft modification, by Warcraft fans for Warcraft fans. You need a copy of the original Warcraft. Warcraft is a registered trademark of Blizzard Entertainment.

Development

War1gus is an open source software. It is developed at GitHub where you can submit bugs, request features and observe progress via issues.
