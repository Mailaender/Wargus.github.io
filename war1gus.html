About War1gus

War1gus is a Warcraft Mod that allows you to play Warcraft with the Stratagus engine. The game looks and sounds exactly like Warcraft, see the screenshot.

Installation

First you need to install Stratagus. During the installation of War1gus you will be asked for a copy of the original Warcraft, so that the installer can extract the data.
